How to use:
	
	create_model.py is the code that must be compiled and run to create a model. 
	create_model.py imports mnist_loader.py and network.py.

	network.py is used to initialize and perform stochastic gradient descent on 
	neural networks.

	mnist_loader.py organizes the data from mnist.pkl.gz into three different data 
	sets for training and testing the model.

	mnist.pkl.gz is the mnist data set containing the data for 70000 handwritten digits.
